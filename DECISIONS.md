# BroPS Decision Log

## D-001 — BroPS is an AI Operating System

**Status:** Accepted

BroPS is not defined as a dashboard or simple chat application. It is a unified AI operating environment.

## D-002 — Bro is the primary operator

**Status:** Accepted

Gev interacts primarily with Bro. Bro coordinates tools, projects, and specialist agents.

## D-003 — Group Chat is first-class

**Status:** Accepted

Group Chat is a core workspace for multi-agent collaboration and must support tasks, decisions, approvals, files, room context, and summaries.

## D-004 — Controlled autonomy

**Status:** Accepted

Agents act only inside explicit scope. Risky, destructive, external, irreversible, financial, or security-sensitive actions require approval.

## D-005 — Repository is canonical

**Status:** Accepted

Approved product and architecture decisions must be recorded in repository documentation. Chat alone is not canonical evidence.

## D-006 — Product model before Debian deployment

**Status:** Accepted

The product, UX, and architecture are designed and validated before server deployment decisions.
