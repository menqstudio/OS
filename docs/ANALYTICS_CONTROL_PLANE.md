# Analytics Control Plane

Bro receives evidence-backed status, not vague progress claims. Four append-only packs own agent intelligence, dashboard engineering, system observability, and portfolio control-room reporting.

## Agent status

Canonical states are `idle`, `running`, `blocked`, `waiting-approval`, `verification`, `failed`, and `completed`. Every status event binds to an immutable `agent_id`, optional `task_id`, observation time, source, and evidence paths.

## Dashboard law

Every KPI has a source, freshness target, owner, thresholds, drill-down, and evidence link. A dashboard without these is invalid.

UI gender metadata accepts only `M` or `F` and is never an authority, routing, skill, evaluation, or verification input.
