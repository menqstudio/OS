# Skills

The Anthropic-compatible structure is fixed: `skills/<skill>/SKILL.md`.

Bro has access to the complete skill library. Specialists do not load all skills by default. They load their core skills plus additional skills required by the task. The canonical 42-skill inventory is in `skills/index.json`. Every registered skill body is included in `config/documentation-manifest.json` and remains subordinate to task contracts, canonical identity/authority, runtime capability gates, evidence requirements, and recovery rules.
