# AGENTS.md

The brain of this repository — full context, current state, roadmap, environment gotchas, and safety rules — lives in **[`CLAUDE.md`](./CLAUDE.md)**.

Every AI or human session MUST read, in order:

1. [`CLAUDE.md`](./CLAUDE.md)
2. [`docs/TEAM_PROTOCOL.md`](./docs/TEAM_PROTOCOL.md)
3. [`PROJECT_STATE.md`](./PROJECT_STATE.md)

Work MUST use a dedicated branch and draft PR. Every meaningful change MUST be committed and pushed. Project-state changes MUST update `PROJECT_STATE.md` in the same commit. Merge requires Gev's approval of the exact candidate HEAD.
